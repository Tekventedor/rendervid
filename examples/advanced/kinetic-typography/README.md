# Kinetic Typography

Dynamic text animations with words appearing, moving, and transforming.

## Preview

![Preview](./preview.gif)

[View animated SVG](preview.svg)

## Features

- Multiple animated words with different effects
- Slide, scale, and fade animations
- Color-coded text hierarchy
- Accent line decorations

## Usage

```bash
pnpm run examples:render advanced/kinetic-typography
```

## Inputs

| Input | Type | Required | Default |
|-------|------|----------|---------|
| `word1` | string | Yes | THINK |
| `word2` | string | Yes | BIGGER |
| `word3` | string | Yes | CREATE |
| `word4` | string | No | TOGETHER |
| `tagline` | string | No | Innovation starts with imagination |
