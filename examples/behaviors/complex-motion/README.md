# Complex Motion

> Complex Motion. Physics or behavioral simulation demonstrating movement patterns.

*Run `node examples/render-all.mjs` to generate the preview GIF and MP4.*

| Property | Value |
|----------|-------|
| **Resolution** | 1920 × 1080 |
| **Duration** | 5s |
| **FPS** | 30 |
| **Output** | Video (MP4) |

## Usage

```bash
# Render this example
node examples/render-all.mjs "behaviors/complex-motion"

# Or render all examples
node examples/render-all.mjs
```
