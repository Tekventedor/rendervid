# Blog Promo

Blog post promotion card with featured image, title, and CTA.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render content/blog-promo
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `category` | string | "TECHNOLOGY" |
| `title` | string | "10 Tips for Better Code" |
| `author` | string | "Jane Developer" |
| `cta` | string | "Read More →" |
