# News Headline

Breaking news style announcement with bold typography.

## Preview

![Preview](preview.gif)

[View animated SVG](preview.svg)

## Usage

```bash
pnpm run examples:render content/news-headline
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `breaking` | string | "BREAKING NEWS" |
| `headline` | string | "Major Tech Company Announces Revolutionary Product" |
| `subtitle` | string | "Industry experts predict massive market impact" |
