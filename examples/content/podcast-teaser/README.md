# Podcast Teaser

Episode preview with audio waveform visual effect.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render content/podcast-teaser
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `showName` | string | "Tech Talk Daily" |
| `episodeTitle` | string | "The Future of AI" |
| `episodeNumber` | string | "Episode 142" |
| `duration` | string | "45 min" |
