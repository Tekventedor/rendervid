# Quote Card

Eye-catching quote card for sharing inspirational quotes on social media.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render content/quote-card
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `quote` | string | "The only way to do great work is to love what you do." |
| `author` | string | "Steve Jobs" |
