# Discount Reveal

Animated discount code reveal with suspense and excitement.

## Preview
![Preview](./preview.gif)

## Usage
```bash
pnpm run examples:render ecommerce/discount-reveal
```

## Inputs
| Input | Type | Required | Default |
|-------|------|----------|---------|
| `offerLabel` | string | No | EXCLUSIVE OFFER |
| `discountValue` | string | Yes | 25% OFF |
| `discountCode` | string | Yes | SAVE25 |
| `validity` | string | No | Valid until December 31st |
