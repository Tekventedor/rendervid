# Flash Sale

High-energy flash sale announcement with countdown timer and urgency messaging.

## Preview
![Preview](./preview.gif)

[View animated SVG](preview.svg)

## Usage
```bash
pnpm run examples:render ecommerce/flash-sale
```

## Inputs
| Input | Type | Required | Default |
|-------|------|----------|---------|
| `discount` | string | Yes | 50% |
| `productName` | string | Yes | ALL ELECTRONICS |
| `timer` | string | Yes | 02:45:30 |
| `cta` | string | No | SHOP NOW |
