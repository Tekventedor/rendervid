# Product Comparison

Side-by-side feature comparison of two products or plans.

## Preview
![Preview](./preview.gif)

[View animated SVG](preview.svg)

## Usage
```bash
pnpm run examples:render ecommerce/product-comparison
```

## Inputs
| Input | Type | Required | Default |
|-------|------|----------|---------|
| `header` | string | Yes | Choose What's Right for You |
| `product1Name` | string | Yes | Basic Plan |
| `product1Features` | string | No | 5 Users • 10GB Storage • Email Support |
| `product2Name` | string | Yes | Pro Plan |
| `product2Features` | string | No | Unlimited Users • 100GB • Priority Support |
