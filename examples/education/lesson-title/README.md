# Lesson Title

Chapter or lesson title card for educational videos.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render education/lesson-title
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `chapter` | string | "Chapter 3" |
| `title` | string | "Understanding Variables" |
| `duration` | string | "15 min" |
