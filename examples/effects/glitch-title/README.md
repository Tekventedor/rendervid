A dramatic cyberpunk-style title intro with advanced glitch effects.

## Overview

This template creates an edgy, tech-inspired title card using multiple glitch effect types. The title appears with a powerful RGB split effect while the subtitle features scramble distortion, creating a dramatic reveal perfect for gaming videos, tech content, or any project needing a cyberpunk aesthetic.

## Features

- **RGB Split Glitch**: Dramatic chromatic aberration on the main title
- **Scramble Effect**: Chaotic distortion on the subtitle
- **Dark Gradient Background**: Deep purple-black gradient
- **Grid Overlay**: Subtle tech-style grid pattern
- **Animated Accent Bars**: Growing horizontal bars with glow
- **Corner Brackets**: Futuristic UI-style corner elements
- **Scanline Effect**: Classic CRT monitor aesthetic
- **Vignette**: Subtle darkening at edges for depth

## Customization

### Inputs

- **Main Title** (required): The primary text to display (e.g., "SYSTEM BREACH")
- **Subtitle** (optional): Secondary text below title (e.g., "Access Granted")
- **Accent Color** (optional): Color for glows and accents (default: #00ff00 - neon green)

### Suggested Color Schemes

**Classic Cyberpunk**
- Accent: `#00ff00` (neon green)
- Look: Matrix-inspired, hacker aesthetic

**Neon Pink**
- Accent: `#ff00ff` (magenta)
- Look: Outrun, retrowave vibes

**Electric Blue**
- Accent: `#00ffff` (cyan)
- Look: Tech, futuristic, sci-fi

**Warning Red**
- Accent: `#ff0000` (red)
- Look: Alert, danger, critical system

## Technical Details

- **Duration**: 6 seconds (180 frames at 30fps)
- **Resolution**: 1920x1080 (Full HD)
- **Glitch Types Used**: RGB Split, Scramble
- **Effects**: GradientText, GlitchEffect

### Timeline

- Frame 0-15: Grid and background visible
- Frame 15: Main title appears with RGB split glitch
- Frame 30: Corner brackets fade in
- Frame 45: Subtitle appears with scramble glitch
- Frame 0-180: Glitches occur periodically throughout

## Animation Behavior

**Glitch Frequency**
- Title: Glitches 0.2 times per second (moderate)
- Subtitle: Glitches 0.15 times per second (occasional)

**Glitch Intensity**
- Title: 0.9 (very strong RGB split)
- Subtitle: 0.6 (moderate scramble)

## Tips

1. **Keep title short**: 1-3 words work best for maximum impact
2. **Contrast colors**: Use bright accent colors against dark background
3. **Font weight**: Bold, heavy fonts enhance the glitch effect
4. **Uppercase**: All caps maintains the tech aesthetic
5. **Testing**: Preview with different accent colors to find your style
