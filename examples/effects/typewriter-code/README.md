# Typewriter Code

Code typing effect with VS Code-style dark theme and matrix-style green cursor.

## Preview

![Preview](./preview.gif)

## Features

- Authentic VS Code dark theme styling
- Window chrome with macOS-style buttons
- Line numbers with proper styling
- Multiple code lines with syntax-inspired colors
- Matrix-style green cursor animation
- File title display in titlebar
- Status bar with language indicator
- Sound pulse on final line
- Professional code editor aesthetic

## Usage

```bash
pnpm run examples:render effects/typewriter-code
```

## Inputs

| Input | Type | Required | Default |
|-------|------|----------|---------|
| `codeLine1` | string | Yes | function calculateSum(a, b) { |
| `codeLine2` | string | Yes | return a + b; |
| `codeLine3` | string | Yes | } |
| `codeLine4` | string | Yes | console.log(calculateSum(5, 3)); |
| `title` | string | No | main.js |
| `cursorColor` | color | No | #00ff41 |

## Color Scheme

This template uses VS Code Dark+ theme colors:
- Background: `#1e1e1e`
- Editor: `#252526`
- Functions: `#dcdcaa` (yellow)
- Strings: `#ce9178` (orange)
- Methods: `#4ec9b0` (cyan)
- Cursor: `#00ff41` (matrix green)

## Design Notes

The template recreates the familiar VS Code interface that developers recognize worldwide. The matrix-style green cursor adds a distinctive hacker aesthetic while maintaining professional code editor styling. Perfect for technical content creators and developer advocates.
