# Conference Intro

Professional speaker introduction card for conferences, panels, and presentations.

## Preview
![Preview](./preview.gif)

[View animated SVG](preview.svg)

## Usage
```bash
pnpm run examples:render events/conference-intro
```

## Inputs
| Input | Type | Required | Default |
|-------|------|----------|---------|
| `conferenceName` | string | No | TECH SUMMIT 2024 |
| `speakerName` | string | Yes | Michael Roberts |
| `speakerTitle` | string | Yes | Chief Technology Officer |
| `company` | string | No | Innovate Labs |
| `sessionTitle` | string | No | The Future of Cloud Architecture |
