# Event Countdown

Countdown timer display for upcoming events, launches, or announcements.

## Preview
![Preview](./preview.gif)

## Usage
```bash
pnpm run examples:render events/event-countdown
```

## Inputs
| Input | Type | Required | Default |
|-------|------|----------|---------|
| `eventTitle` | string | Yes | BIG SALE EVENT |
| `days` | string | Yes | 07 |
| `hours` | string | Yes | 12 |
| `minutes` | string | Yes | 30 |
