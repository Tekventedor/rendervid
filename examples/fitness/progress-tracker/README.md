# Progress Tracker

Weekly or monthly fitness progress visualization.

## Preview

![Preview](preview.gif)

[View animated SVG](preview.svg)

## Usage

```bash
pnpm run examples:render fitness/progress-tracker
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `title` | string | "WEEKLY PROGRESS" |
| `metric` | string | "Workouts Completed" |
| `current` | string | "5/7" |
| `percentage` | string | "71%" |
