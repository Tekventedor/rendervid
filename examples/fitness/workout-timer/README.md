# Workout Timer

Exercise interval timer with workout name and countdown.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render fitness/workout-timer
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `exerciseName` | string | "JUMPING JACKS" |
| `duration` | string | "00:45" |
| `round` | string | "Round 3 of 5" |
| `nextUp` | string | "Next: Push-ups" |
