# Daily Special

Today's special promotion with limited-time offer messaging.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render food/daily-special
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `badgeText` | string | "TODAY ONLY" |
| `day` | string | "TUESDAY SPECIAL" |
| `dishName` | string | "Gourmet Burger & Fries" |
| `price` | string | "$14.99" |
