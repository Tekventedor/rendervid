# Menu Item

Restaurant dish showcase with price.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render food/menu-item
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `dishName` | string | "Grilled Salmon" |
| `description` | string | "Fresh Atlantic salmon..." |
| `price` | string | "$24.99" |
| `category` | string | "CHEF'S SPECIAL" |
