# Price Drop

Real estate price reduction announcement.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render real-estate/price-drop
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `badge` | string | "PRICE REDUCED" |
| `address` | string | "456 Oak Avenue" |
| `oldPrice` | string | "$599,000" |
| `newPrice` | string | "$549,000" |
| `savings` | string | "Save $50,000!" |
