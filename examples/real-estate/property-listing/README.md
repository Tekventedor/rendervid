# Property Listing

Real estate property showcase with key details and price.

## Preview

![Preview](preview.gif)

## Usage

```bash
pnpm run examples:render real-estate/property-listing
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `address` | string | "123 Maple Street" |
| `price` | string | "$549,000" |
| `beds` | string | "4" |
| `baths` | string | "3" |
| `sqft` | string | "2,450" |
