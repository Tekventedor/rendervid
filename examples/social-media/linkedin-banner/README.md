# LinkedIn Banner

Professional banner for LinkedIn company pages and profiles.

[View animated SVG](preview.svg)

## Features

- LinkedIn banner dimensions (1584x396)
- Professional gradient design
- CTA button
- Wide format optimized

## Inputs

| Key | Type | Required | Default |
|-----|------|----------|---------|
| `companyName` | string | Yes | "Your Company" |
| `tagline` | string | No | "Building the future..." |
| `ctaText` | string | No | "We're Hiring!" |
| `primaryColor` | color | No | #0a66c2 |
| `backgroundColor` | color | No | #1b1f23 |

## Quick Start

```bash
pnpm run examples:render social-media/linkedin-banner
```
