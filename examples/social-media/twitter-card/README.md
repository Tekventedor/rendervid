# Twitter Card

Image optimized for Twitter/X cards and posts.

## Preview

![Preview](preview.gif)


![Preview](preview.png)

## Features

- Twitter card dimensions (1200x628)
- Clean professional layout
- Handle integration
- Optimized for engagement

## Inputs

| Key | Type | Required | Default |
|-----|------|----------|---------|
| `headline` | string | Yes | "Big News!" |
| `body` | string | No | "We have something exciting..." |
| `handle` | string | No | "@yourbrand" |
| `primaryColor` | color | No | #1d9bf0 |
| `backgroundColor` | color | No | #15202b |

## Quick Start

```bash
pnpm run examples:render social-media/twitter-card
```
