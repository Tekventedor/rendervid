# YouTube Thumbnail

Eye-catching thumbnail optimized for YouTube videos.

## Preview

![Preview](preview.png)

## Features

- 16:9 aspect ratio (1280x720)
- Bold typography
- Badge for emphasis
- High contrast design

## Inputs

| Key | Type | Required | Default |
|-----|------|----------|---------|
| `title` | string | Yes | "10 TIPS" |
| `subtitle` | string | No | "You Need to Know" |
| `badge` | string | No | "NEW" |
| `primaryColor` | color | No | #ef4444 |
| `backgroundColor` | color | No | #18181b |

## Quick Start

```bash
pnpm run examples:render social-media/youtube-thumbnail
```
