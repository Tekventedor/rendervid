# Highlight Intro

Dynamic intro for gaming highlights and montages.

## Preview

![Preview](preview.gif)

[View animated SVG](preview.svg)

## Usage

```bash
pnpm run examples:render streaming/highlight-intro
```

## Inputs

| Key | Type | Default |
|-----|------|---------|
| `title` | string | "EPIC MOMENTS" |
| `gameName` | string | "Apex Legends" |
| `episode` | string | "Highlights #42" |
